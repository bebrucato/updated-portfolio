Projects

boozyTunes

This fun project was a collaboration with classmates. boozyTunes utilizes a cocktail recipe API to provide the user with some exciting new beverage options. The other cool part about this app is that it also generates Spotify playlists based on the user's listening preferences. Although... if you're feeling a bit more daring, there is a randomize option accessible via the "Party Time!" button and the app will generate a random cocktail and playlist for you.

Check it out!
Weather Dashboard

Being able to check weather on the go is always important. With this app, you can search ANY city in the world and view its current temperature, humidity, wind speed and UV index. This app also displays the 5-day forecast of your selected city, which makes it easier to plan ahead.

Check it out!
The Quiz App

Much like the title suggests, this is a quiz app. In this quiz, your obscure horror film and novel knowledge will be tested. With every wrong answer, you'll lose 10 seconds off the timer, so you better be quick!
